Neste trabalho, desenvolvi um programa que simule a persistência de registros em um arquivo de dados (.DAT), 
considerando as restrições de armazenamento em blocos e diferentes estratégias de organização de registros.
O objetivo é compreender as implicações práticas das decisões de projeto relacionadas ao tamanho dos registros, 
limitação de blocos e métodos de alocação de dados em disco.
